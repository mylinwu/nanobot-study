# 04-4｜工具性能优化：批处理、缓存与并发控制

## 1. 常见瓶颈

- 单条请求逐个调用工具，吞吐低。
- 重复请求未缓存，成本高。
- 无并发限制，导致下游被打挂。

## 2. 三个高收益优化

1. 批处理（batch invoke）
2. 结果缓存（TTL + key 设计）
3. 并发门控（semaphore / rate limit）

## 3. 伪代码

```python
with semaphore(8):
    result = cached_call(tool, args, ttl=600)
```

## 4. 练习

为搜索工具设计缓存键：至少包含 query、topk、语言、时间窗。
