# 06-2｜模型路由与结构化输出落地

## 1. 路由目标

在质量与成本之间找到平衡。

## 2. 路由策略

- planning：高推理模型
- extraction：低成本模型
- final_review：高质量模型

## 3. 动态升级策略

先用低成本模型，质检失败再升级。

## 4. 结构化输出

- JSON Schema 校验
- 失败自动 repair（限次）

伪代码：

```python
out = model.run(prompt)
if not schema.validate(out):
    out = model.run(repair_prompt(out))
```

## 5. 练习

为“任务报告”定义 schema（结论、证据、风险、计划），并写 3 条必填校验。
