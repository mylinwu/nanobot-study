# NanoBot 学习文档（中文教学版）

> 本教程针对“想真正看懂 Agent 项目实现”的学习者设计，不是概览大纲，而是**可落地的渐进式教程**。

## 学习目标

1. 搭建完整心智模型：用户请求如何变成可执行任务。
2. 理解关键模块存在的“工程原因”，而不是背名词。
3. 掌握模块从 V1 到 V4 的演进逻辑（每个模块至少三版）。
4. 能对照源码定位：入口、计划、执行、记忆、评估、运维。
