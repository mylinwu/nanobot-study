# 03-4｜控制面故障演练：超时、重试风暴与熔断

## 1. 典型故障

- 下游工具抖动，导致步骤连续超时。
- 无上限重试，造成队列堆积。
- 并发放大导致系统雪崩。

## 2. 防护顺序

1. timeout
2. retry with backoff
3. circuit breaker
4. replan / fallback

## 3. 演练脚本思路（教学）

- 人工注入 30% timeout。
- 观察 `FAILED->RETRYING->BLOCKED` 转移。
- 统计恢复时间与失败率。

## 4. 验收标准

- 不出现无限重试。
- P95 不持续恶化。
- 故障结束后队列可自动回落。
