# 05-1｜记忆系统：短期、长期与注入策略

## 1. 三层记忆

- 短期：当前任务中间结论。
- 会话：当前会话偏好与约束。
- 长期：跨会话用户画像。

## 2. 注入原则

- 相关才注入（避免噪音）。
- 新事实优先于旧偏好。
- 注入过程可解释（记录用了哪条记忆）。

## 3. 记忆摘要

```python
summary = llm.summarize(history[-8:])
memory.save(session_id, summary, ttl='7d')
```

## 4. 常见风险

- 摘要漂移（越摘要越失真）。
- 过期记忆污染当前任务。
