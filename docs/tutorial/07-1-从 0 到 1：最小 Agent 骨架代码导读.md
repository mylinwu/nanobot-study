# 07-1｜从 0 到 1：最小 Agent 骨架代码导读

## 1. 最小链路

`request -> planner -> executor -> response`

## 2. 最小代码骨架

```python
def handle(goal):
    plan = planner.make(goal)
    outputs = []
    for s in plan:
        outputs.append(executor.run(s))
    return synthesize(outputs)
```
